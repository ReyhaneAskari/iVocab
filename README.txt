
A mobile synonym learning app, targeted specifically for iPhone and 
Android browsers.

How to build iVocab:

1. Install Apache ant.
2. Download Java Appengine SDK.
3. Download GWT 2.0 or later.
4. Add the correct path information for the Appengine SDK, and GWT 2.0 
   to 'iVocab/build.properties'.
5. Drop to the command line, and in the iVocab directory type:
   'ant'.


To run iVocab locally, type:
'ant runserver'
